**Final-Project-MA-615**

This repository contains the final project for MA 615. The objectives of this project are to:

- Demonstrate skills and knowledge in building a webpage using Shiny

- Demonstrate skills and knowledge in creating a geospatial map

- Demonstrate skills and knowledge in building interactive tables

- Demonstrate skills and knowledge in creating statistical plots

- Demonstrate skills and knowledge in cleaning and imputing missing data

**Project Files**

The project includes the following key files:

- Final-project.Rproj: Used to open and manage the project in RStudio.
  
- Final-report-pdf.pdf: PDF version of the report

- Final-report-MA615.html: The final project report rendered in HTML format.

- Final-report-MA615.qmd: The Quarto source file containing the report code and step-by-step documentation of how app.R was built.

- app.R: Contains the Shiny application code for the interactive webpage.

- supplementary-table-s1-3.csv: The dataset used for this project.

- lemur_icon.png: The icon used in the HTML report to link to the webpage. At the end of the report, please click the Lemur icon to access the webpage.

- README.md file: Provides explanations and instructions on how to use the uploaded files and folders.

**Project Folders**

The repository also contains two folders:

- www/: Contains image files for the lemur species used in the Shiny application.

- rsconnect/: Stores deployment-related files used to connect app.R to shinyapps.io.
